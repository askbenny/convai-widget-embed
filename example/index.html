Welcome to the AskBenny Widget Example!

This page demonstrates how to use the AskBenny widget, which is a custom wrapper for ElevenLabs ConvAI widgets. The widget below will load when you provide a valid ElevenLabs agent ID.

üìù Basic HTML Usage

Simply include the script and use the custom element:

<!-- Load the widget --> <script src="https://unpkg.com/@askbenny/widget"></script> <!-- Use the widget --> <ask-benny agent-id="your-agent-id"></ask-benny>
